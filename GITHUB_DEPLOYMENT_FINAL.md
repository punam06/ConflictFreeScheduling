## 🚀 **Usage Instructions**

### **Quick Start**
```bash
# Clone the repository
git clone https://github.com/punam06/ConflictFreeScheduling.git
cd ConflictFreeScheduling

# Build the project
mkdir build && cd build
cmake ..
make

# Run comprehensive scheduling
./scheduler --comprehensive-routine --algorithm graph-coloring

# View generated PDF
open output/academic_schedule_comprehensive_routine.pdf
```

### **Advanced Usage**
```bash
# Test all algorithms
./scheduler --run-all

# Generate enhanced reports
./scheduler --comprehensive-routine --enhanced-output

# Database operations
sqlite3 data/scheduling.db "SELECT COUNT(*) FROM schedule;"
```
